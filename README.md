SmartOrder Manager

Sistema de gestión de pedidos basado en el concepto de listas.

Tecnologías:
- Typescript
- Express
- HTML
- CSS
- Javascript

Funcionalidades:

1 Crear pedidos
2 Ver lista de pedidos
3 Aprobar pedidos

El sistema utiliza listas para almacenar los pedidos y simular el flujo del diagrama de procesos.
