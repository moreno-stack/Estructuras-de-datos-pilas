# ⚡ Quick Start - Comenzar en 2 Minutos

## Instalación Rápida

### 1️⃣ Instalar dependencias
```powershell
pip install -r requirements.txt
```

### 2️⃣ Ejecutar la aplicación
```powershell
streamlit run app/frontend/streamlit_app.py
```

¡Listo! Se abrirá en `http://localhost:8501`

---

## Primeros Pasos

1. **Abre la app** en tu navegador
2. **Ve a "📦 Crear Pipeline"**
3. **Selecciona un template** (ej: "completo")
4. **Ve a "▶️ Ejecutar"**
5. **Haz clic en "🚀 EJECUTAR PIPELINE"**
6. **¡Observa cómo se ejecuta!**

---

## Atajos

| Acción | Comando |
|--------|---------|
| Ejecutar | `streamlit run app/frontend/streamlit_app.py` |
| Con otro puerto | `streamlit run app/frontend/streamlit_app.py --server.port=8502` |
| Ejecutar tests | `python -m pytest tests/` |
